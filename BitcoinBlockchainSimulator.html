Conceptos clave

Bloque
Paquete de transacciones confirmado por la red.
Hash
Huella digital del bloque. Si cambia un dato, cambia el hash.
Nonce
Número que el minero prueba miles de veces para encontrar un hash válido.
Prev Hash
Hash del bloque anterior. Une toda la cadena.
Merkle Root
Resumen hash de todas las transacciones del bloque.
Fee
Pequeña comisión que incentiva al minero a incluir transacciones.

Esta simulación mantiene la lógica pedagógica central de Bitcoin: transacciones → mempool → bloque → hash → nonce → prev hash → validación. Se han simplificado varios detalles técnicos reales para que sea visual y usable en clase.
